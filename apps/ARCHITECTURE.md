# Product recommendation: best service format

## Best mode of service

**Primary wedge:** VS Code extension  
**Primary system of record:** Web app / SaaS dashboard

### Why not only a browser extension?
A browser extension is weak for scanning private repos on disk and for understanding package manifests, env files, server code, and multi-language source trees.

### Why not only a VS Code extension?
You can scan repos well from VS Code, but billing connectors, org auth, dashboards, alerting, and benchmarking belong in a SaaS product.

### Final answer
Use both, but lead with **VS Code extension + SaaS**.
